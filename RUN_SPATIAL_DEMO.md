# Run Spatial Visualization Demo 🎮

## Quick Start

### 1. Install Pygame (if not already installed)
```bash
pip3 install pygame
```

### 2. Run the Demo
```bash
python3 test_spatial_visualization.py
```

---

## What You'll See

A **2D GTA-style world** with:
- **30 agents** moving around a small city
- **Locations**: Homes, offices, park, stores, gym, coffee shop
- **Real-time movement**: Agents walk through continuous space
- **Mental health colors**:
  - 🟢 **Green**: Thriving
  - 🔵 **Blue**: Stable
  - 🟡 **Yellow**: Struggling
  - 🔴 **Red**: Crisis

---

## Controls

### Camera
- **WASD** or **Arrow Keys**: Move camera
- **Mouse Wheel**: Zoom in/out
- **Middle Mouse + Drag**: Pan camera

### Interaction
- **Left Click**: Select agent (shows details panel)
- **SPACE**: Pause/Resume simulation
- **L**: Toggle location labels
- **P**: Toggle agent paths
- **ESC**: Quit

---

## UI Elements

### Top Left
- Current simulation time
- FPS counter
- Agent count
- Mental health summary (counts by category)

### Bottom Left
- Controls help

### Right Side (when agent selected)
- Agent name, age, role
- Mental health stats
- Current position
- Movement status
- Recent actions

---

## What's Happening

Agents are:
- **Moving continuously** through 2D space (not teleporting)
- **Following paths** generated by A* pathfinding
- **Picking random destinations** when they arrive
- **Updating mental health** based on their state

You can:
- **Watch them move** in real-time
- **See mental health** as colors
- **Click any agent** to see their full details
- **Pan and zoom** to explore the world

---

## Technical Details

**World Size**: 800m x 600m
**Agent Speed**: 1.5-2.5 m/s
**Time Scale**: 60x (1 real second = 1 sim minute)
**Target FPS**: 60
**Pathfinding**: A* algorithm with 2m grid resolution

## Troubleshooting

**Pygame not found?**
```bash
pip3 install pygame
```

**Low FPS?**
- Reduce number of agents in `test_spatial_visualization.py`
- Disable path rendering (press P)
- Disable labels (press L)

**Agents not moving?**
- They should pick new destinations automatically
- Check console for errors
