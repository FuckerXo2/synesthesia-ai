# Synesthesia: Complete System Overview 🌍

**"The Sims meets Black Mirror" - A living mental health simulation**

---

## What Is Synesthesia?

A **real-time population mental health simulator** with 5,000-10,000 AI agents living in a 2D world. Each agent is a unique person with:
- Deep backstory, values, fears, goals
- Mental health that fluctuates realistically
- Relationships that evolve
- Memories of their experiences
- Real conversations with emotional impact

**Not a clinical tool** - it's a **living world** where you can:
- Watch mental health patterns emerge across a population
- Zoom from macro (heatmaps) to micro (individual conversations)
- Talk to any agent and hear their story
- Query the world with an Oracle AI

## Key Features

### 1. **LLM-Powered Intelligence**
- **Population Generation**: Describe a society, LLM generates realistic demographics
- **Society Orchestration**: LLM creates roles, events, stressors, support systems
- **Agent Identities**: Each agent has unique backstory generated by LLM
- **Conversations**: Realistic dialogue with internal thoughts
- **Emergent Events**: LLM generates unexpected events based on population state

### 2. **Real-Time Continuous Simulation**
- **Not turn-based**: Agents live continuously like NPCs in games
- **Actions have duration**: Sleep 8 hours, work 1 hour, walk 5 minutes
- **Proximity-based interactions**: Conversations trigger when agents are nearby
- **Time scale**: Configurable (1 real second = 60 sim seconds)

### 3. **Deep Psychological Simulation**
- **Mental Health States**: Anxiety, depression, stress, wellbeing
- **4 Categories**: Thriving, Stable, Struggling, Crisis
- **Dynamic Changes**: Mental health fluctuates based on actions, events, conversations
- **Memory System**: Agents remember experiences with emotional impact
- **Resilience**: Positive memories buffer against trauma

### 4. **Rich Social Dynamics**
- **Relationships**: Parent, child, spouse, sibling, friend, coworker
- **Relationship Quality**: Trust, affection, respect
- **Evolution**: Bonds strengthen through positive interactions
- **Automatic Conversations**: Agents talk when nearby (30% chance for friends, 5% for strangers)
- **Shared Memories**: Conversations create memories for both agents

### 5. **2D Spatial World**
- **Continuous Space**: Agents move through 2D world (not grid-based)
- **A* Pathfinding**: Navigate around obstacles
- **Real-Time Visualization**: 60 FPS pygame rendering
- **Mental Health Colors**: Green (thriving), Blue (stable), Yellow (struggling), Red (crisis)
- **Interactive**: Click agents, pan camera, zoom in/out

### 6. **Emergent Behavior**
- **No hardcoded scenarios**: LLM generates everything from user description
- **Realistic societies**: Tech startups, medieval villages, space stations, etc.
- **Dynamic events**: Recurring (weekly meetings) and random (crises, celebrations)
- **Cascading effects**: Events trigger other events
- **Population-level patterns**: Mental health trends emerge naturally

## Quick Start

### 1. Install Dependencies
```bash
pip3 install -r requirements.txt
```

### 2. Set Up Environment
```bash
# Copy .env.example to .env
cp .env.example .env

# Add your NVIDIA API key
# LLM_API_KEY=nvapi-YOUR_KEY_HERE
```

### 3. Run Tests

**Test 2D Visualization** (30 agents, interactive):
```bash
python3 test_spatial_visualization.py
```

**Test Real-Time Simulation** (5 agents, conversations):
```bash
python3 test_quick_conversation_sim.py
```

**Test Identity Generation** (10 unique agents):
```bash
python3 test_identity_generation.py
```

**Test Conversations** (2 agents talking):
```bash
python3 test_conversations.py
```

### 4. Run Full Simulation
```bash
# Generate population
python3 synesthesia_population_generator.py

# Run simulation
python3 run_simulation.py
```
