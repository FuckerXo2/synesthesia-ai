## Step-by-Step

### 1. Start the Web App
```bash
python3 web_app.py
```

### 2. Open Browser
Go to: **http://localhost:5001**

### 3. Create Simulation
- Type society description (e.g., "High school")
- Set population (50-100)
- Click "GENERATE & GO"

### 4. Wait for Generation
You'll see:
- "Generating society structure..."
- "Creating spatial world..."
- "Creating population..."

### 5. Simulation Starts
You'll see:
- **Left side**: Canvas with agents moving
- **Right side**: Stats panel

### 6. Find Oracle AI
**Scroll down in the right panel** to see:

```
┌─────────────────────┐
│ 🔮 Oracle AI        │
│ Ask anything about  │
│ the simulation      │
│                     │
```

### 7. Ask Questions

**Option A: Click Quick Query Buttons**
- Click any of the 4 buttons
- Wait 2-5 seconds
- See results below

**Option B: Type Custom Question**
- Click in the input box
- Type your question

## Quick Reference

### Location
- **Screen**: Simulation screen (after generation)
- **Panel**: Right side panel
- **Position**: Below "Population Stats", scroll down
- **Section**: "🔮 Oracle AI"

### Components
1. **Title**: "🔮 Oracle AI"
2. **Subtitle**: "Ask anything about the simulation"
3. **Quick Queries**: 4 buttons (Most at Risk, Trends, Role Analysis, Interventions)
4. **Custom Input**: Text box + "Ask" button
5. **Results Area**: Shows answers, stats, insights, recommendations

### Interaction
- **Click button** → Instant query
- **Type + Enter** → Custom query
- **Type + Click Ask** → Custom query
- **Wait 2-5 seconds** → See results
- **Scroll in results** → See full response

## Troubleshooting

### "I don't see Oracle AI"
- Make sure simulation is running (not setup screen)
- Scroll down in the right panel
- Look for "🔮 Oracle AI" heading

### "Nothing happens when I click"
- Check browser console (F12) for errors
- Make sure simulation is running
- Wait 5-10 seconds for first query (cold start)

### "I see 'AI service temporarily unavailable'"
- API is overloaded
- Wait 30 seconds and try again
- System will automatically try different models

### "Results are cut off"
- Scroll within the results box
- Results box has its own scrollbar

## Tips

1. **Start with quick queries** to see how it works
2. **Then try custom questions** for specific needs
3. **Be specific** in your questions for better answers
4. **Ask for numbers** to get quantitative data
5. **Ask "why"** to get deeper analysis
